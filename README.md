## Postup pro výběr kávy
1. Vyberte typ kávy na obrazovce.
2. Nastavte sílu kávy a množství cukru
3. Stiskněte tlačítko "Start" a počkejte na dokončení přípravy.
4. Užijte si vaši kávu!
